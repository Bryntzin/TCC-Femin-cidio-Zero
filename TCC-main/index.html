sobre o site



Nosso trabalho de Conclusão de Curso desenvolvido por Pedro Correa, João Fernando, Bryan Miguel e Marcos Vinicius utiliza o Desenvolvimento de Sistemas para estruturar uma plataforma informativa sobre o combate ao feminicídio. O projeto foca na organização de dados e no fornecimento de orientações claras, servindo como uma base de conhecimento acessível e segura. A arquitetura do sistema prioriza a disseminação de informações críticas, como estatísticas reais, legislações vigentes e canais oficiais de apoio. Através de uma interface intuitiva, o software organiza conteúdos educativos que auxiliam na identificação de sinais de violência e na compreensão dos direitos das vítimas. O desenvolvimento une técnica e consciência social, transformando a programação em um veículo de informação essencial para a sociedade.
